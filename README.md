# OutputTest
MicroPython Signal Controller for Ian Hart's Pico02 / Pico03 Raspberry Pi Pico based CBUS Board (16 outputs) for MERG CBUS

The code takes the signal ID (within the scope of the Pico board) and aspect from an ON event, ad sets the outputs accordingly
